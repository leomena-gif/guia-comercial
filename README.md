# Guía Comercial Local - Proyecto Final UTN

Aplicación web dinámica desarrollada como proyecto final del curso **Programador Web Inicial - Front End Developer** de la UTN BA.

## Tecnologías utilizadas

- **Node.js** — Entorno de ejecución del servidor
- **Express.js** — Framework web
- **Handlebars (HBS)** — Motor de plantillas
- **MySQL** — Base de datos relacional
- **Nodemailer + Mailtrap** — Envío de correos en entorno de pruebas
- **Express-session** — Manejo de sesiones
- **dotenv** — Variables de entorno
- **Bootstrap 5** — Framework de estilos
